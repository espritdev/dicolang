À propos de Dico'Lang

Dico'Lang est un dictionnaire multilingue moderne qui combine la puissance de plusieurs services de traduction et de définition pour vous offrir une expérience complète d'apprentissage des langues.

Fonctionnalités

  • Traduction instantanée
  • Définitions détaillées
  • Support de multiples langues
  • Interface moderne et responsive
  • Mode sombre/clair

Technologies

  • Flask (Backend Python)
  • Bootstrap 5 (Interface utilisateur)
  • Wiktionary API (Définitions)
  • Google Translate API (Traductions)
